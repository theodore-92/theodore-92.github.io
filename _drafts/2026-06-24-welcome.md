Jekyll과 Hydeout 테마로 블로그를 시작합니다.

## 이미지 넣는 법

이 블로그는 이미지를 직접 경로로 박지 않고, 아래처럼 `img.html` include를 통해서만 불러옵니다.

```liquid
{% raw %}{% include img.html name="2026-06-24-jekyll/cover.png" alt="표지 이미지" caption="시작하는 마음으로" %}{% endraw %}
```

이렇게 해두면 나중에 이미지 호스팅을 다른 곳(예: Cloudflare R2)으로 옮길 때도 `_config.yml`의 `image_base_url` 한 줄만 바꾸면 모든 글이 한꺼번에 따라옵니다. 실제 이미지 파일은 `assets/images/posts/2026-06-24-jekyll/` 폴더에 넣어두면 됩니다.

## 댓글

댓글은 우선 비활성 상태입니다. Isso 서버를 띄운 뒤 `_config.yml`에 `isso_url`만 채워주면 바로 작동합니다.
